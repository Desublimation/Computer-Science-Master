/* Problem 1 [4pt]:
The following are two basic predicates for list manipulation:
my_first/2 and my_last/2. We may refer to a predicate by writings it
as name/arity; hence, my_first/2 means a predicate named my_first with
two arguments.

my_first(X,Y) succeeds if X is the first element of list Y.
my_last(X,Y) succeeds if X is the last element of list Y.

Write definitions for my_first and my_last.
*/

/* Problem 1 Answer: */
my_first(X, [X|_]).

my_last(X, [X]).
my_last(X, [_|T]) :-
    my_last(X, T).

/* Problem 2 [4pt]:
   Write a predicate is_decreasing(X) that succeeds if X is a list of
   decreasing numbers -- Each number is either the same or lower than
   the preceding number.
*/

/* Problem 2 Answer */
is_decreasing([]).
is_decreasing([_]).
is_decreasing([A,B|T]) :-
    A >= B,
    is_decreasing([B|T]).

/* Problem 3 [4pt]:
   Write a predicate element_at(X,Y,N) that succeeds if X is the Nth 
   element of list Y. Y is 0-indexed.

   NOTE: Don't worry about the error cases: i.e, N greater than the 
   length of Y.  
*/

/* problem 3 Answer */
element_at(X, [X|_], 0).
element_at(X, [_|T], N) :-
    N > 0,
    N1 is N - 1,
    element_at(X, T, N1).

/* Problem 3 Test: */
%:- element_at(3,[1,2,3],2).   % SUCCEED
%:- element_at(1,[1,2,3],0).   % SUCCEED

%:- element_at(1,[1,2,3],1) -> fail ; true.     % FAIL

/* Problem 4 [4pt]:
   Write a predicate insert_at(E,Y,N,Z) that succeeds if Z is the list Y 
   with E inserted at index N -- Insert X at index N in Y.

   NOTE: Don't worry about the error cases: i.e, N greater than the 
   length of Y.  
*/

/* Problem 4 Answer */
insert_at(E, Y, 0, [E|Y]).
insert_at(E, [H|T], N, [H|Z]) :-
    N > 0,
    N1 is N - 1,
    insert_at(E, T, N1, Z).

/* Problem 6 [4pt]:
   Write a predicate sumlist(List,Sum) which succeeds if Sum is the total value
   of all the elements of List. This will be a top down recursion.
   The recursion clause will add the current value to the result of the sum
   of the rest of the list.
   We have already provided the base case for this predicate underneath
   'Problem 6 Answer'. You just need to add the recursive clause.
*/

/* Problem 6 Answer */
sumlist([], 0).
sumlist([H|T], Sum) :-
    sumlist(T, Rest),
    Sum is H + Rest.


/* Problem 6 Test */
%:- sumlist([], 0).
%:- sumlist([], 1) -> fail ; true.
%:- sumlist([1,2,3,4], 10).
%:- sumlist([1], 1).

/* Problem 7 [4pt]:
   Write the predicate sumlist2(List,Sum) which succeeds if Sum is the sum total
   of all the elements of List. Instead of adding the current value to the
   result of the sum of the tail, you will calculate the partial sum of the all
   the elements you have reached so far. You will need an extra argument to
   store the partial sum, so you will write an auxilitary predicate sumlist2/3
   to handle the extra argument.
*/

/* Problem 7 Answer */

sumlist2(List, Sum) :-
    sumlist2(List, 0, Sum).

sumlist2([], Acc, Acc).
sumlist2([H|T], Acc, Sum) :-
    Acc1 is Acc + H,
    sumlist2(T, Acc1, Sum).

/* Problem 7 Test */

%:- sumlist2([], 0).
%:- sumlist2([], 1) -> fail ; true.
%:- sumlist2([1,2,3,4], 10).
%:- sumlist2([1], 1).
